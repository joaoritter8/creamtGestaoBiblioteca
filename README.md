## Tecnologias
- C#
- ASP.NET MVC 5
- JavaScript

## Requisitos
- .NET Framework 6.0 ou maior
- SQL Server

## Sobre a aplicação

- Com essa aplicação é possível cadastrar categorias de livros, livros, usuários e autores e gerenciar os empréstimos e devoluções dos livros através do CRUD.
